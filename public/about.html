About Done

Done is an opinionated productivity application inspired by the Cult of Done Manifesto. It embraces the philosophy of getting things done quickly and moving forward, rather than getting stuck solving a problem that does not want to be fixed.

How to Use Done

Done categorizes your tasks into four states:

  • Not Knowing: The Created State
  • Action: Tasks you're currently working on. They appear at the top because they are most important.
  • Done: Completed tasks.
  • Destroyed: Tasks that are no longer relevant or have been abandoned.

When you create a task, it goes into the Not Knowing category. The timer starts, giving you 2 weeks to get the task done. If 2 weeks is not enough, try breaking the task into smaller pieces.

To update a task's status, just click the button corresponding to where you need to move it. An Action can be changed to a Done or Destroyed task. And Not Knowing to all 3. Once a task is Done or Destroyed there is no where else for it to go.
